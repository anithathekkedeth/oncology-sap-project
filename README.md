# oncology-sap-project
This project demonstrates end-to-end statistical analysis for a Phase II oncology trial.
## Key Components
- Statistical Analysis Plan (SAP)
- Tables, Listings, Figures (TLFs)
- Kaplan-Meier analysis (PFS)
- Subgroup analysis (forest plot)
- Bayesian hierarchical modeling

## Highlights
- Implemented estimand framework (ICH E9 R1)
- Demonstrated handling of intercurrent events
- Integrated frequentist and Bayesian methods
- Created submission-style outputs

## Tools Used
- R (survival, rstanarm, ggplot2)
- Report generation (PDF)
